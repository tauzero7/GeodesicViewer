OpenGL window

You can switch between then OpenGL 3D and the Draw 2D views by clicking on the tabs or by pressing 'Ctrl 1' or 'Ctrl 2', respectively.

Keyboard handle 3D-View

Key Description
x Set eye onto x-axis
y Set eye onto y-axis
z Set eye onto z-axis
Page-Up Decrease distance by 5.0
Page-Down Increase distance by 5.0
Up Decrease distance by 0.5
Down Increase distance by 0.5
w Toggle between solid and wireframe representation

Mouse handle 3D-View

Button Description
left orbit around point-of-interest (poi); local z-axis
middle move point-of-interest (poi) along z-axis
middle + ctrl move point-of-interest (poi) parallel to camera view
left + ctrl orbit around point-of-interset (poi); global z-axis
right zoom in/out

Keyboard handle 2D-View

Key Description
r Reset viewing range
c Center origin
s Set aspect ratio to 1

Mouse handle 2D-View

Button Description
left move origin
left + ctrl select a region to zoom in (top-left to bottom-down)
or out (bottom_down to top-left)
right zoom in/out
right + ctrl zoom in/out with origin fixed
