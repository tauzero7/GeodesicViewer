File

The File menu has the following entries:
Name Description
Reset Reset all parameters.
Load Setting Load configuration file.
Save Setting Save configuration file.
Load ViewParams Load viewing parameters file.
Save ViewParams Save viewing parameters file.
Reset ViewParams Reset viewing parameters file.
Load all Load configuration, viewing parameters, and objects file.
Save Image 2D Save OpenGL-2D view.
Save Image 3D Save OpenGL-3D view.
Write Protocoll Write protocoll of the current setting.
Quit Quit program.

Selecting Write Protocoll opens the dialog:


The protocoll saves the following details:
  • Parameters: the parameters of the metric, the geodesic integrator, and the initial conditions (schw_orbit_1.ini).
  • ViewParameters: the viewing parameters like the eye position, line colors, background colors, projection type,... (schw_orbit_1.cfg).
  • Objects: the objects list (schw_orbit_1.obj).
  • Geodesic: the points and the tangents of the geodesic in metric coordinates (schw_orbit_1.points).
  • 3d Image: the current OpenGL-3D content (schw_orbit_1.3d.png)
  • 2d Image: the current OpenGL-2D content (schw_orbit_1.2d.png)
  • Report: the report of the metric with the current settings (schw_orbit_1.rep).

Objects

The Objects menu has the following entries:
Name Description
Load object file Load object file.
Append object file Append object file.
Save object file Save object file.
Show objects Show all objects

Selecting Show objects opens the object dialog which is a simple text editor.


Each object must be defined in a specific syntax. To add e.g. a three-dimensional sphere to the OpenGL environment, you should do the following:
  • Select an object with Add object (The syntax of the object appears in the Syntax line.)
  • Press + button to add the object into the text field.
  • Adjust the parameters of the sphere.
  • Apply commits the object list to the OpenGL environment.

To close the dialog, press Close.
To remove a single object, erase it from the text field.
To remove all objects, press Remove all.
